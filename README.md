# Event Hiring Frontend

This is the frontend of the Event Hiring application built using **Next.js**.

## Features
- Multi-step form (Event Details → Hire Type → Requirements → Preview)
- Dynamic form based on hire type (Planner / Performer / Crew)
- Review before submission
- Sends data to backend API
- Clean and simple UI

## Tech Stack
- Next.js
- React
- Tailwind CSS

## Running Locally
1. Install dependencies:
   npm install

2. Start development server:
   npm run dev

3. Open in browser:
   http://localhost:3000
